البدء السريع

مرحبًا بك في توثيق Raven information. سيساعدك هذا الدليل على البدء باستخدام منصتنا والاستفادة القصوى من ميزاتها.

إنشاء حساب

لبدء استخدام Raven information، تحتاج أولاً إلى إنشاء حساب:

  1. انقر على زر "إنشاء حساب" في الزاوية العلوية اليمنى
  2. املأ النموذج بالمعلومات المطلوبة
  3. اضغط على زر التسجيل
  4. ستتلقى رسالة تأكيد على بريدك الإلكتروني

ملاحظة: يمكنك استخدام المنصة بدون حساب ولكن مع قيود على عدد الطلبات.

تسجيل الدخول

بعد إنشاء الحساب، يمكنك تسجيل الدخول باستخدام:

  1. البريد الإلكتروني وكلمة المرور
  2. حساب Google (قريبًا)
  3. حساب GitHub (قريبًا)

واجهة الدردشة

واجهة الدردشة هي المكان الرئيسي للتفاعل مع الذكاء الاصطناعي. إليك كيفية استخدامها:

إرسال رسالة

لإرسال رسالة إلى المساعد الذكي:

  1. اكتب سؤالك أو رسالتك في مربع النص
  2. اضغط على زر الإرسال أو Enter
  3. انتظر حتى يتم توليد الرد

تنسيق الرسائل

يدعم المساعد الذكي تنسيقات مختلفة للرسائل:

  • الأسئلة: "ما هو الطقس اليوم في الرياض؟"
  • الأوامر: "اكتب لي رسالة بريد إلكتروني للاعتذار عن التأخير"
  • الشفرة البرمجية: "أكتب دالة جافاسكريبت لجمع عناصر مصفوفة"
// مثال على طلب لكتابة كود برمجي اكتب لي دالة بايثون لحساب معدل الأعداد في قائمة

تحميل الملفات

يمكنك تحميل ملفات مختلفة لتحليلها من قبل الذكاء الاصطناعي:

الملفات المدعومة

  • مستندات: PDF, Word (.doc, .docx), Excel (.xls, .xlsx)
  • ملفات نصية: .txt
  • ملفات برمجة: .js, .lua, .html, .css
  • ملفات مضغوطة: .zip, .rar

كيفية التحميل

  1. انقر على زر "رفع ملف"
  2. اختر الملف من جهازك
  3. انتظر حتى يكتمل التحميل
  4. اطرح أسئلة حول محتوى الملف

تحذير: الحد الأقصى لحجم الملف هو 10MB للخطة المجانية و50MB للخطط المدفوعة.

التحكم الصوتي

يدعم Raven information التفاعل الصوتي من خلال الميكروفون:

إعداد الميكروفون

  1. تأكد من توصيل الميكروفون
  2. اضغط على زر السماح عند طلب الإذن
  3. اضغط على زر الميكروفون في واجهة الدردشة

الأوامر الصوتية

يمكنك استخدام الأوامر التالية:

  • "اكتب رسالة إلى [الاسم]"
  • "ابحث عن معلومات عن [الموضوع]"
  • "حلل الملف الذي حملته"
  • ("توقف" لإيقاف التسجيل)

تكامل API

يمكنك دمج Raven information في تطبيقاتك باستخدام واجهة برمجة التطبيقات (API):

الحصول على مفتاح API

  1. سجل الدخول إلى حسابك
  2. انتقل إلى صفحة الإعدادات
  3. اختر "مفاتيح API"
  4. انقر على "إنشاء مفتاح جديد"

مثال استخدام

// مثال باستخدام JavaScript const response = await fetch('https://api.raveninfo.ai/v1/chat', { method: 'POST', headers: { 'Authorization': 'Bearer YOUR_API_KEY', 'Content-Type': 'application/json' }, body: JSON.stringify({ model: 'mistral-7b', messages: [{ role: 'user', content: 'مرحبا' }] }) }); const data = await response.json(); console.log(data.choices[0].message.content);

مشاكل شائعة

الرد يأخذ وقتًا طويلاً: قد يكون الخادم مشغولاً، حاول مرة أخرى لاحقاً.
